# mane-kelsa-app
Android application connecting domestic workers with local users.

## Project Overview

Mane Kelsa is an Android application designed to help users manage household workers efficiently. The application allows users to add, view, and maintain worker information through a simple and user-friendly interface.

## Features

* Add new worker details
* View list of registered workers
* Store and manage worker information
* User-friendly Android interface
* Easy navigation between screens

## Technologies Used

* Kotlin
* Android Studio
* XML Layouts
* Android SDK
* Gradle

## Project Structure

Main files included in the project:

* MainActivity.kt
* AddWorkerActivity.kt
* Worker.kt
* WorkerRepository.kt
* WorkerAdapter.kt
* DataHolder.kt
* AndroidManifest.xml
* XML Layout Files

## How to Run

1. Download or clone the project.
2. Open the project in Android Studio.
3. Sync Gradle dependencies.
4. Connect an Android device or start an emulator.
5. Run the application.
